Este é um programa simples, de entrada e saída de dados, com objetivo de treinar a sintaxe Java. 

O objetivo é simular um banco que solicita Nome, Agência, Conta e Saldo que deseja sacar. 

No final, aparece uma mensagem escrita "Olá, [nome de cliente], agência [nº da agência], conta [nº da conta] o saldo de [valor] já está disponível para saque".
